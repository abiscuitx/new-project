# FlyingStudio
1. 项目文件夹为
2. 深度分析项目文件夹，不要关联其他项目文件夹。
3. 按下列要求修改项目文件夹下的：.gitignore,LICENSE,README.md。
4. .gitignore，保留原内容基础上，添加系统、编辑器、缓存、密钥、敏感、语言等通用忽略规则，参考示例文件内容：.gitignore.example。
5. LICENSE，备份原证书，添加Apache License 2.0许可证文本，时间为当前年份，作者为FlyingStudio，参考示例文件内容：LICENSE.example。
6. README.md，将原有内容删除并移动在文档最下方## 原有内容中，生成新内容严格按照示例文件结构：README.md.example，
7. 使用gitRemove.sh删除项目文件夹中已被.gitignore忽略的文件，参考命令：/gitRemove.sh --apply ' 项目文件夹路径 '。
